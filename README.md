# React Markdown Viewer

### A markdown viewer built with ReactJS

## Live at:

- https://mkdown.netlify.app

## Features:

- [x] Headings 1-6
- [x] Bullet Points
- [x] Numbered bullets
- [x] Links
- [x] Image Links
- [x] Code Snippets & Boxes
- [x] Checkboxes
- [x] Horizontal Rules
- [x] Bold/Italics - other formats
- [x] Block quotes
- [x] Tables
